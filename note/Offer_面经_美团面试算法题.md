一组数中如果只有一个数是1个，其他数是3个，怎么找出这个数？
http://blog.csdn.net/ojshilu/article/details/14762791

```java
int singleNumber(int A[], int n) {  
        int ones = 0;  
        int twos = 0;  
        int i;  
        for(i=0;i<n;i++)  
        {  
            ones = (~twos) & (ones ^ A[i]);  
            twos = (~ones) & (twos ^ A[i]);//注意这里用的ones是新的ones，这一点和数电不同。  
        }  
        return ones;  
}  
```
